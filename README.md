# Skyrim Effect

**Skyrim Effect** is a library designed to enable users to utilise [Effect-TS](https://effect.website/) for
developing [Skyrim](https://en.wikipedia.org/wiki/The_Elder_Scrolls_V:_Skyrim) mods
using [Skyrim Platform](https://github.com/skyrim-multiplayer/skymp/blob/main/skyrim-platform/README.md).

⚠️ **Early Development Stage**: Please note that this project is in its very early stages of development and is not yet
suitable for actual use. Contributions and feedback are welcome to help steer the development process!
